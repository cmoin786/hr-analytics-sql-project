# Employee Management SQL Analytics Project

## Project Overview

This project simulates a real-world Employee Management System using PostgreSQL.

The database tracks employees, departments, attendance, projects, and employee project allocations. Advanced SQL queries are used to generate HR insights, salary analysis, attendance reports, and project performance metrics.

---

## Database Schema

### Tables

1. Departments
2. Employees
3. Attendance
4. Projects
5. Employee_Project

## Business Questions Solved

### Employee Analysis

* Total employees
* Highest and lowest salaries
* Average salary
* Salary ranking
* Annual salary calculation
* Salary quartiles

### Department Analysis

* Department-wise employee count
* Department-wise average salary
* Highest salary by department
* Department project budget

### Attendance Analysis

* Attendance reports
* Absent employee tracking
* Average working hours
* Attendance performance reporting

### Project Analysis

* Employee project allocation
* Total project hours
* Project contribution percentage
* Multiple project assignments

### Advanced SQL Analysis

* Salary ranking using RANK()
* LAG() and LEAD() functions
* Running totals
* Cumulative salary calculations
* Department-wise salary comparison
* CTE-based top salary analysis
